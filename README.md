# Otachaki

## App for training Go territory counting and position evaluation

Live Demo: https://siasio.github.io/otachaki/
For an Android app, download an .apk file from the Releases tab and enable installing from unknown sources on your device.

## Features

- **Multiple Training Modes**: Practice score calculation on 9x9, 13x13 and 19x19 final positions, and score estimation on middle-game 19x19 positions
- **Interactive Go Board**: Realistic stone rendering with animated timer and visual feedback
- **Custom Datasets**: Create personalized training sets with configurable timing and scoring
- **Statistics Tracking**: Monitor your progress across different position types
- **Keyboard Shortcuts**: Arrow keys for quick navigation
- **Cross-Platform**: Runs on Android, Web, and Desktop
